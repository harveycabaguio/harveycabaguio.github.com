The elementary Project

elementary is an organization focused on creating apps for the Linux desktop that are designed to be usable and beautiful. elementary OS is the accumulation of all this work into one cohesive operating system. It has been downloaded over 3 million times and is translated in 20+ languages.

I've been with the elementary team since after the first release of elementary OS back in 2011, and have been helping guide some of the design decisions in it ever since.

Desktop Design

Multitasking View – The place to manage your workspaces/desktops and windows much like OSX's mission control or Windows 10's virtual desktops.

Icons

I've also created some of the app icons found in elementary OS. These are handcrafted in multiple sizes using Inkscape to ensure pixel perfection in all the places that the OS will use these icons.

Website

For elementary OS's third release I created the initial designs of the website refresh. Our focus for this refresh is to:

  • make a minimum-viable-product for distributing elementary OS
  • be mobile first
  • allow for continuous integration (constant small improvements)
  • gracefully degrade in older browsers
