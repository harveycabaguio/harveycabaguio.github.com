Working with Links

Creating links is incredibly easy, it does not matter if its a link to another webpage, or to a page within your site, or if its an email or file.

The <a> tag

This tag allows you to turn anything inside it into a link. But just placing text within <a> </a> won't do the trick. The tag needs the href attribute for the link to go anywhere.

Code

<a href="https://www.google.ca/">Google</a>

Result

Google

As you can see, the a tag tells the browser that the content within is a link, and the href attribute of the a tag tells the browser where that link will go.

There are other attributes like target which allow you to change how the link is opened. For example:

Code

<a href="https://www.google.ca/" target="_blank">Google</a>

Result

Google

target="_blank" will make your link open in a new tab.

Absolute Links

This type of link is best for linking to external websites. You must include the protocol at the beginning of the link (http://).

Code

<a href="https://www.google.ca/">Google</a>

Result

Google

Relative Links

This type of link is used to link other pages within your website.

Linking to a file in the same folder

<a href="index.html">Working with Links</a>

Linking to a file in folder inside the current one

<a href="folder/index.html">Example</a>

Linking to a file one level above the current folder

<a href="../me.png">My Picture</a>
