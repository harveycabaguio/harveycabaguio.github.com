Website Design & Development

This course introduces students to the basics of front-end web development. Creating HTML elements such as text, links, and images and manipulating their styles using CSS are some of the topics covered in this course.

Topics Covered:

  • Research and Design
  • Basic HTML
  • Cascading Style Sheet
  • Color
  • Manipulating Text
  • Manipulating Links
  • Manipulating Images
  • Setting the Page Structure
  • Modifying the Page Layout
  • Creating Lists
  • Using Tables
  • Using Forms
