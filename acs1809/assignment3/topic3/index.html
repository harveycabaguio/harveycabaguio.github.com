Creating Lists

There are many types of lists. Here is an unordered list:

Code

          
            <ul>
              <li>item 1</li>
              <li>item 2</li>
              <li>item 3</li>
            </ul>

Result

  • item 1
  • item 2
  • item 3

As you can see, ul tells the browser that the li in it are going to be unordered, and thus styled each li to have a bullet point beside it.
