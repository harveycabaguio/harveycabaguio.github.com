Working with Images

Adding images to your webpage is very similar to adding links, albeit with small differences

img tag

Code

<img src="../me.png" width="120px" height="120px">

Result

Just like a, img tells your browser you are about to add an image. The attribute src is similar to href, and follows the same rules for linking absolute/relative links. The height and width attributes define the dimensions that the image will be displayed on the webpage.
